# Tic Tac Toe Game

This is a simple Tic Tac Toe game built using HTML, CSS, and JavaScript.

## Features
- Two player game
- Interactive UI
- Winning detection
- Reset game option

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Play
1. Player O starts the game
2. Players take turns placing X and O
3. First player to align 3 marks wins
